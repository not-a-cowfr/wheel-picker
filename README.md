# wheel-picker

Simple cli tool to make and pick from a wheel.

## Installing

from source

```sh
git clone https://github.com/not-a-cowfr/wheel-picker
cargo install --path wheel-picker
```

from cargo

```sh
cargo install wheel-picker
```

from [github releases]

[github releases]: https://github.com/not-a-cowfr/wheel-picker/releases/latest
